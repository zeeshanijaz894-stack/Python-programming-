A modern, responsive, and professional website for Excellence Academy school.

## Features

✨ **Modern Design**
- Clean and professional layout
- Gradient backgrounds and smooth animations
- Fully responsive design for all devices

📱 **Responsive Navigation**
- Mobile-friendly hamburger menu
- Smooth scrolling navigation
- Active link highlighting

🎓 **Key Sections**
- **Home**: Eye-catching hero section with call-to-action
- **About**: School mission and key features
- **Academics**: Comprehensive academic programs
- **Admissions**: Application process and inquiry form
- **Contact**: Location, contact info, and map integration

🎨 **Design Elements**
- Professional color scheme
- Animated statistics counter
- Interactive program cards
- Smooth transitions and hover effects
- Font Awesome icons integration

⚡ **Interactive Features**
- Form validation and submission handling
- Intersection observer for scroll animations
- Mobile menu toggle
- Scroll-to-top button
- Keyboard navigation support

## Customization Tips

1. **Change Colors**: Update the CSS variables in `:root`
   ```css
   --primary-color: #your-color;
   --secondary-color: #your-color;
   --accent-color: #your-color;
   ```

2. **Update Content**: Replace all placeholder text with your school information

3. **Add Images**: Add your school logo and background images

4. **Modify Google Map**: Replace the embed code in the contact section

5. **Update Forms**: Add your backend form handler or email service
